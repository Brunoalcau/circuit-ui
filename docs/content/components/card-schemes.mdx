# Card Schemes

<Status.InReview/> <Status.Description size={Text.KILO}>Likely to be moved to the icon library</Status.Description>

Card scheme icons indicate the accepted card schemes. The icons are displayed based on a given list of scheme IDs.

It can be used in forms with the `CardNumberInput` component.

    iconIds={['mastercard', 'visa', 'ec', 'googlePay', 'applePay']}
