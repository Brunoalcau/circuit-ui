# Inline notification

<Status.InReview/> <Status.Description size={Text.KILO}>Likely to be renamed</Status.Description>

Inline notifications lets you display notifications to merchants within the content flow.
It is generally used in form validation.

You can use the [Message](/components/message#notification-messages) component to display notifications as toasts or banners, removed from the content flow.
